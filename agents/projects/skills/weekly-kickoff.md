# SKILL: weekly-kickoff

## Trigger
Weekly kickoff, Monday planning, week priorities, what should I work on

## Steps
1. Check momentum data: any active project with 7+ days of silence is stale — name it first
2. Propose top 3 priorities for the week, weighted by project priority (High > Medium) and staleness
3. Give each priority a concrete first step doable today, not a theme
4. Respect the domain split: dairy operation runs on its own rhythm — flag it separately, never rank it against tech projects

## Output Format
- Stale projects first (skip the section if none)
- Numbered top 3 priorities, each with a first step
- Under 200 words total

## Edge Cases
- Everything is stale: don't shame — pick the single highest-leverage restart
- No momentum data at all: base priorities on projects.md alone and suggest logging updates
